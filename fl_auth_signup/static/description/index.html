Auth Signup Form

This module add phone number, address and birth date to auth sign up page

Setting

Go to Settings > General Settings > Users.

In Customer Account - select option Free sign-up (B2C).

It allows your new users to get a sign-up option.

Fields

  • Phone Number
  • Address Line 1
  • Address Line 2
  • City
  • Zip
  • State
  • Country
  • Birth Date
