SO & PO Multi Product Selection

This module allows select multi-product in Sale order and Purchase order

Sale Order

Go to Sales > Orders > Quotations.

Select Products button in quotation/sale order form view.

When you click Select Products button wizard will open for select products.

Click on Add an item then open wizard for select products and then press the select button.

selected products added in wizard and press Add button.

Selected products are automatically added in order lines.

Purchase Order

Go to Purchases > Purchase > Requests for Quotation.

Select Products button in RFQ/purchase order form view.

When you click Select Products button wizard will open for select products.

Click on Add an item then open wizard for select products and then press the select button.

Selected products added in wizard and press Add button.

Selected products are automatically added in order lines.

Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
