Quick Task

This module allow to create quick task for project

Project

Go to Project > Search > Tasks.

Click on create button.

Task Form

Enter task title, project, deadline date, planned hours.

List of Tasks

List of tasks in kanban view.

Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
